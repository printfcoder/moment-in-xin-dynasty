# 参考资料

1. 《三家注史记》卷8引《史记集解》：皇甫谧曰：“高祖以秦昭王五十一年生，至汉十二年，年六十二。”
2. 颜师古等注《汉书》卷1下《高帝纪下》：臣瓒曰：“帝年四十二即位，即位十二年，寿五十三。”
